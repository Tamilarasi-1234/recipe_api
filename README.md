Description:
This project is a Recipe Management System built using Python, Flask, and SQLite. Users can:

Browse a large collection of recipes (1000+ entries).

Search recipes by name or cuisine.

View detailed recipe info including ingredients, instructions, and nutrition facts.

REST API ready for integration with any frontend or mobile app.

Features:

Database Connected: Recipes stored in SQLite (recipes.db).

Search Functionality: Quickly find recipes by keyword or cuisine.

Interactive Frontend: Easy-to-use interface with HTML/CSS/JS.

REST API: Provides recipe data in JSON format for external apps.

Technologies Used:

Python 3.x

Flask

SQLite

HTML, CSS, JavaScript
